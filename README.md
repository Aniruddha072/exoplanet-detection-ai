# Exoplanet Detection using AI/ML

This repository contains a data pipeline and deep learning architecture to detect exoplanetary transits using time-series stellar-flux data

##🚀 Getting Started

1. **Clone this repository** to your local computer
2. **Download the dataset.** Download the raw zip file from our shared storage: https://drive.google.com/file/d/1wpda_oZ_Leo7GqxBK3JlRotYmSs8MWW8/view?usp=sharing
3. **Extract the files.** Unzip the folder and place 'exoTrain.csv' and 'exoTest.csv' directly inside your local 'data/raw/' directory
4. **Run the pipeline.** Open the project workspace in VS code and execute the notebook.
